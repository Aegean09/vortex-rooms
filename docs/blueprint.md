# **App Name**: Vortex

## Core Features:

- Session Creation: Generate a shareable link upon clicking 'Create Room' to invite users to a new voice and text chat session.
- Username Authentication: Prompt users for a username upon joining a session; no database required, temporary username for the session.
- User Presence: Display a list of users currently active in the voice/text chat session.
- Voice Chat: Enable real-time, peer-to-peer voice communication between users within a session using web sockets.
- Text Chat: Enable real-time text communication between users within a session using web sockets.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) to give a modern, energetic feel.
- Background color: Dark gray (#28282B) for a sleek, lightweight design.
- Accent color: Light Purple (#BE95FF) to complement the primary blue.
- Body and headline font: 'Inter' sans-serif for a modern and neutral look, ensuring readability and a lightweight feel.
- Use minimalist, outlined icons for session controls and user actions.
- Design a clean, single-page layout with a prominent session link sharing area.
- Use subtle transitions and animations for user presence updates and message notifications.
